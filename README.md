# DELAYED.

> "The quieter you become, the more you can hear."

**DELAYED.** is a philosophical anti-impulse engine designed to combat reactive digital communication. It is a communication interface that forces a "cooling-off period" on every message, post, or thought before it is allowed to reach the world. 

The application uses biometric-style heuristics (typing speed, backspace frequency, and sentiment analysis) to calculate a "Volatility Index," which determines exactly how long a user must wait before their words become permanent.

### 2. The Cooling Zone

* **Regret Debt Multiplier:** Frequent cancellations cause the system to "glitch," increasing all future delay timers as the system learns you are an unreliable narrator of your own intentions.
